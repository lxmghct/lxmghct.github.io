# 1. 隐藏中英文切换图标

weasel输入法在切换中英文的时候会在光标处显示一个中英文切换图标，有时候这个图标会挡住光标，影响输入体验。旧版本只能通过修改源码来隐藏这个图标，根据这个[issue](https://github.com/rime/weasel/issues/520)
修改下面这个文件即可：
[RimeWithWeasel.cpp](https://github.com/rime/weasel/blob/master/RimeWithWeasel/RimeWithWeasel.cpp#L404-L407)


在新版本的weasel输入法中，可以通过配置文件来隐藏这个图标，修改`weasel.custom.yaml`文件，添加下面这行配置即可：
```yaml
patch:
  show_notifications: false
```

# 2. Shift 键切换中英文
之前一直用的搜狗输入法，使用 Shift 切换中英文时会直接把当前的拼音输入，然后切换到英文输入状态，继续输入英文。weasel可以通过修改 `default.custom.yaml` 文件来实现这个功能，添加下面的配置：
```yaml
patch:
  "ascii_composer/switch_key/Shift_L": commit_code
  "ascii_composer/switch_key/Shift_R": commit_code
```

参考 [https://blog.csdn.net/sdujava2011/article/details/84098971](https://blog.csdn.net/sdujava2011/article/details/84098971)

# 最终配置
输入方案使用了雾凇拼音输入法([rime-ice](https://github.com/iDvel/rime-ice))。

`default.custom.yaml` 文件的最终配置如下：
```yaml
customization:
  distribution_code_name: Weasel
  distribution_version: 0.16.1
  generator: "Rime::SwitcherSettings"
  modified_time: "Sun Sep 29 09:31:23 2024"
  rime_version: 1.11.2
patch:
  "ascii_composer/switch_key/Shift_L": commit_code
  "ascii_composer/switch_key/Shift_R": commit_code
  "menu/page_size": 8
  schema_list:
    - {schema: rime_ice}
```

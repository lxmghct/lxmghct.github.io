我想给手机安装一个开源的邮件客户端 K-9 Mail，通过一个 app 收发多个邮箱的邮件。

qq 邮箱：
- Incoming IMAP Configuration
  * IMAP Server: imap.qq.com
  * Security: SSL/TLS
  * Port: 993
- Outgoing SMTP Configuration
  * SMTP Server: smtp.qq.com
  * Security: SSL/TLS
  * Port: 465

Gmail 邮箱：
- Incoming IMAP Configuration
  * IMAP Server: imap.gmail.com
  * Security: SSL/TLS
  * Port: 993
- Outgoing SMTP Configuration
  * SMTP Server: smtp.gmail.com
  * Security: SSL/TLS
  * Port: 465

但是在配置 Gmail 邮箱时，通过 OAuth 授权成功后，检查收件服务器设置会失败：
```
failed to connect to imap.gmail.com/28.0.0.5(port 993) ...
```
大概率是网络代理没有成功，我在 Clash 里设置了全局代理以及修改了一些其他配置都没有用。我对邮件服务器连接的网络代理不太了解，暂时没有解决。我换了另一个开源邮件应用 FairMail 也是同样的问题。最后换了 Gmail 官方 App。
